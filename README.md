# 📝 Todo App (Fullstack Vue + Laravel)

Aplikasi Todo List berbasis Fullstack menggunakan **Vue.js + Laravel REST API**.

---

## 🚀 Tech Stack

* Frontend: Vue 3 + Axios
* Backend: Laravel
* Database: MySQL
* UI: Custom CSS + Dark Mode

---

## 📁 Struktur Project

```
todo-app/
├── backend/     # Laravel API
├── frontend/    # Vue App
└── README.md
```

---

## ⚙️ Cara Menjalankan Project

### 🔧 Backend

```bash
cd backend
composer install
cp .env.example .env
php artisan migrate
php artisan serve
```

---

### 🎨 Frontend

```bash
cd frontend
npm install
npm run dev
```
